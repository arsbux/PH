Week-by-Week Curriculum

Week 1 — "Vibe & Launch" (UGX 70,000)

Deliverable: Live landing page

  • Session 1: Design fundamentals — layout, spacing, typography, colors. Quick Figma/pen sketch.
  • Session 2: HTML structure + responsive CSS basics (mobile-first). Pair work: build hero + nav.
  • Homework: Complete hero + CTA, push to GitHub, deploy static site (Netlify/Vercel/Glitch).

Why this converts: Visible end-product in 48-72 hours. Tangible proof = parent trust.

Week 2 — "Interaction & Polish"

Deliverable: Interactive multi-section site

  • Session 1: Advanced CSS — transitions, transforms, and simple CSS animations; accessible UX.
  • Session 2: JavaScript basics for interactivity (menu toggles, tabs, form validation).
  • Homework: Add two interactive components, implement simple analytics.

Week 3 — "Polish, Publish, Pitch"

Deliverable: Portfolio-ready project + demo video

  • Session 1: Performance & deployment best-practices. SEO basics + meta and OG tags.
  • Session 2: Demo day prep — script a 60-90s video, record, edit (phone), and write the project case study.
  • Demo: Host a mini-demo day (parents + school) or upload videos for portfolio.
